SentiFrameNet

SentiFramNet (SFN) is an extension to FrameNet that provides information on the presuppositions and entailments of the Lexical Units in FrameNet’s frames.

The SFN extension references FrameNet release 1.6.

A publication introducing the resource is currently under review. We provide a sample of its contents here. (The archive contains a brief README as well.)

For evaluation purposes, we also produced frame-semantically annotated version of 11 documents out of the 70-document MPQA3.0 corpus. For each document, we have the basic FrameNet annotations and a converted version that displays the annotations of the text with the abstract predicates associated with the frame instances in the original version.
